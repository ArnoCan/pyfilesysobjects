Get Current User Name¶

The current logon user name could be retrieved by the call:

1
2
3
import filesysobjects.userdata

filesysobjects.userdata.getcurrent_username()

For complete details refer to [API] / [code].

Get HOME Directory¶

The current logon user home directory could be retrieved by the call:

1
2
3
import filesysobjects.userdata

filesysobjects.userdata.gethome()

For complete details refer to [API] / [code].

Get User HOME Directories¶

The specific HOME directory of an user could be retrieved by the call:

1
2
3
4
import filesysobjects.userdata

filesysobjects.userdata.getdir_userhome()  # default is current user
filesysobjects.userdata.getdir_userhome('userx')

For complete details refer to [API] / [code].
