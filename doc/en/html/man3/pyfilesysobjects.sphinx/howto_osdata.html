Get Directory with OS Configuration¶

The os specific common shared configuration directory could be retrieved by the call:

1
2
3
import filesysobjects.osdata

filesysobjects.osdata.getdir_osconfigdata()

For complete details refer to [API] / [code].

Get Directory with Application Configuration¶

The os specific directory to be used by applications for context specific configuration data could be retrieved by the call:

1
2
3
import filesysobjects.osdata

filesysobjects.osdata.getdir_osappconfigdata()

For complete details refer to [API] / [code].

Get Directory with Application Data¶

The application specific directory for configuration data could be retrieved by the call:

1
2
3
import filesysobjects.osdata

filesysobjects.osdata.getdir_osappdata()

For complete details refer to [API] / [code].
